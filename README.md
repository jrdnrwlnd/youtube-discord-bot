# youtube-discord-bot
Simple discord bot running on [Serenity](https://github.com/serenity-rs/serenity) and [Songbird](https://github.com/serenity-rs/songbird "Songbird")

Usage:
- `--play <youtube URL> or <youtube search query>` play any youtube video with its URL or play the first video from youtube search
- `--leave` bot leaves the voice channel
- 
Dependencies:
- [ffmpeg](https://github.com/FFmpeg/ "ffmpeg")
